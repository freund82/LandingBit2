# Инструкция по установке лендинга на Joomla

## Структура файлов

После распаковки архива вы получите следующую структуру:

```
joomla_landing/
├── assets/
│   ├── css/
│   │   └── styles.css
│   ├── js/
│   │   └── script.js
│   └── images/
│       └── Maldives.jpg
├── components/
│   └── com_landing/
│       ├── proxy.php
│       └── landing.php
└── README_Joomla_Installation.md
```

## Установка на Joomla

### 1. Загрузка файлов

1. Загрузите папку `assets` в корневую директорию вашего сайта:
   - CSS файлы: `/assets/css/styles.css`
   - JavaScript файлы: `/assets/js/script.js`
   - Изображения: `/assets/images/Maldives.jpg`

2. Загрузите компонент в папку компонентов Joomla:
   - `/components/com_landing/proxy.php`
   - `/components/com_landing/landing.php`

### 2. Создание статьи в Joomla

1. Перейдите в Панель управления Joomla
2. Перейдите в "Контент" → "Материалы"
3. Нажмите "Создать"
4. Вставьте содержимое файла `landing.php` в редактор статьи
5. Убедитесь, что включен редактор "Код" (Editor - Code) для правильного отображения HTML

### 3. Настройка меню

1. Перейдите в "Меню" → "[Название вашего меню]"
2. Нажмите "Создать"
3. Выберите тип пункта меню "Материал" → "Один материал"
4. Выберите созданную статью
5. Установите этот пункт как главную страницу (если требуется)

### 4. Настройка proxy.php

Файл `proxy.php` должен находиться в папке компонентов и быть доступен по адресу:
`/components/com_landing/proxy.php`

Убедитесь, что файл имеет права доступа 644.

## Важные замечания

1. Файл `proxy.php` должен быть доступен по адресу, указанному в файле `Прочти меня.txt`
2. Убедитесь, что ваш сервер поддерживает PHP и имеет доступ к внешним API
3. Проверьте, что ваш сервер позволяет выполнение функции `file_get_contents` для внешних URL
4. При возникновении проблем с CORS, возможно потребуется настройка заголовков в .htaccess
